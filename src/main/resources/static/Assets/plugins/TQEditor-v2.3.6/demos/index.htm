开发TQEditor的原因是不满意当前的在线编辑器, 要么功能强大,但体积也超大, 要么体积小了, 功能却不能满足需要.
这是国内第一个兼容IE9的在线编辑器, 也是一个功能体积比最优的的在线编辑器. 为你提供一个功能强大, 体积小, 全浏览器兼容的在线编辑器.
目标: 最好用的在线编辑器

当前最新版 v2.0.0 stable 正式版 与 2.1.0 beta版
2.1.0版更新内容:

1. UBB插件点击查看
2. 完善事件 (oninit, onfocus, onblur, onkeydown, onkeyup, ongetcontent, onsetcontent) 点击查看
3. 扩展的接口(提取图片/Flash/flv, 删除链接/图片/Flash/Object对象) 点击查看
4. 皮肤优化, 插件优化

特点：

  1. 小身材,JS(56k)+CSS(8k)+图标(7k) 总计71K, 启用gz压缩后不到30K
  2. 全浏览器支持, 当前支持 IE6, 7, 8, 9, chrome 8+, FF 3+, Opera 10+, Safari5+
  3. Word代码清理,有效清理从word中粘贴的代码同时保留格式
  4. 默认使用SWFUpload组件处理上传, 用户体验超好
  5. 自带源码格式化
  6. 支持皮肤功能, 只需修改一个CSS即可自定义皮肤
  7. 一键自动排版功能,内置自动图文排版, 默认即适合大多数的文章内容排版形式
  8. 灵活的扩展功能, 方便按业务需要扩展功能
  9. 可视化的UBB编辑功能
  10. 方便的事件,接口调用
未来特性, 具体实现时间未定:
  1. 屏幕截图 (需要插件支持, 且只支持IE)
  2. 直接粘贴Word文档的图片 (需要插件支持, 且只支持IE)
  3. 表格功能增强(方便的设置表格./行/列/单元格属性, 合并单元格等)
  4. 其它语言包
